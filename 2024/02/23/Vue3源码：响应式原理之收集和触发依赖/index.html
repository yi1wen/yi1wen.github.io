Vue3源码：响应式原理之track和trigger依赖

发表于 2024-02-23 | 分类于 源码 ， Vue3

track

track 总结

  1. 装依赖的结构是三层，第一层 WeakMap，第二层 Map，第三层 Set (桶)

  2. 第三层是 Dep， Set 类型装 effect

  3. dep 收集 effect 是用于触发依赖，effect 收集 dep 是为了更新依赖

trigger

trigger 总结

  1. 根据触发类型，对需要触发的 Dep 处收集 effect，扁平化再包装为一个 Dep 准备触发
  2. 触发的顺序是先 computed 再 watch 和 render
  3. 触发的方式是有 scheduler 就 scheduler 没有就 fn
