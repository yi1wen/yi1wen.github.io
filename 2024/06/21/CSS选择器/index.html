CSS选择器

发表于 2024-06-21

优先级

!important > 内联 > #ID > .类别 class > 属性[] > 伪类 > 元素 > 伪元素 > 通配符 * > 继承

选择器语法

  1. 选择器1选择器2：没有空格，交集
  2. 之间, 并集

属性选择器

  1. [属性=属性值]
  2. [属性^=属性值] 开头
  3. [属性$=属性值] 结尾
  4. [属性*=属性值] 包含

关系选择器

  1. 父> 第一代子元素
  2. 空格 祖先 后代
  3. 兄 + 第一个弟，紧跟在兄后的元素的弟
  4. 兄 - 所有弟

动态的类

  1. :link 链接没有被访问过
  2. :visited
  3. :hover
  4. :active 点击
  5. :focus
  6. :first-child 第一个子元素 :last-child
  7. :nth-child ?????
  8. :nth-last-child
  9. :nth-of-child
  10. :first-of-type 父元素的第一个同类型的子元素 :last-of-type
  11. :only-child 唯一一个子元素
  12. :only-of-type
  13. :empty 没有任何内容的元素
  14. :not 过滤

伪元素

创建一些不在DOM树中的元素，querySelector无法获取

  1. ::before
  2. ::after
  3. ::first-line 用于块级元素 第一行
  4. ::first-letter 用于块级元素 第一个字母
  5. ::selection 鼠标框选的内容

伪元素的要点：

  1. window.getComputedStyle(element, ‘:before’) 能获取到伪元素的css对象
  2. 默认 display:inline, 所以无法自定义宽高，可以设置 display: block/ inline-block

伪元素的修改：

  1. 使用 class 修改，removeClass，addClass
  2. 插入css规则，使用 document.styleSheet[0].insertRule(‘div::before {}’,index) 或 let style = document.createElement(“style”), style.insertRule, document.head.appendChild(style)
  3. attr()，应用Dom元素的data属性，dom元素设置 data-key=value，用 element.dataset.key 修改

getPropertyValue 和直接用键值访问css属性的区别

  1. float属性：键为 cssFloat 和 styleFloat
  2. 键值访问：驼峰写法
